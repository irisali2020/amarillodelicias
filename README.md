AmarilloDulce

Es un proyecto con propósitos educativos para aprender a usar React, desarrollando una SPA similar a una tienda con esquema ecommerce, la cual cuenta con un carrito de compras.

Para su desarrollo se ha utilizado  React con distintos componentes, contextos, rutas protegidas,formularios. Se trabajó con MockApi, la cual sirve para simular el Backend y además Bootstrap y Toastify, para mejorar todo el estilo de la SPA.

El administrador del sistema tiene la facilidad para crear, leer, actualizar y eliminar los productos que son promocionados a traves de la SPA.

El cliente podria comprar los productos que desee asi como pagar (simulación) estando en un ambiente protegido ya que debe utilizar un token para acceder a esta facilidad.
